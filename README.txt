# Happy Moments CRM

## PASOS PARA USAR EN GITHUB PAGES

1. Crear un repositorio nuevo en GitHub.
2. Subir el archivo index.html.
3. Ir a:
   Settings → Pages
4. En Source seleccionar:
   Deploy from branch
5. Seleccionar:
   Branch = main
   Folder = /root
6. Guardar.

GitHub te dará una URL como:
https://usuario.github.io/repositorio

---

## CONFIGURAR ULTRAMSG

1. Abrir UltraMSG.
2. Copiar:
   - Instance ID
   - Token

3. Pegarlos en:
   Configuración UltraMSG

4. Guardar configuración.

---

## IMPORTANTE

GitHub Pages NO puede ejecutar automatizaciones 24/7 si la página está cerrada.

Para automatización real necesitas:
- Node.js
- Render
- VPS
- Railway
- Cron jobs
